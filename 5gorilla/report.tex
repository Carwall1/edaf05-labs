  \section{Results}

  The check script passed all test cases. The largest inputs run in about 2-3s. Most execution time is spent in the double loop populating the 2D array. Backtracking and parsing time are almost instant.

  \section{Implementation details}

  I implemented an iterative, bottom-up dynamic programming solution. I used a 2D array to cache maximum alignment scores and a nested dictionary for O(1) substitution matrix lookups. 
  
  The time complexity is O(N x M), where N and M are the string lengths. This is because I visit each cell in the N x M grid exactly once, performing constant-time additions and max comparisons.
